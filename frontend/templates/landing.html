How it works

01

Connect GitHub

Sign in with your GitHub account. RepoGator only requests read access to your profile — no broad permissions upfront.

02

Add a repository

Pick a repo. RepoGator installs a webhook on it so GitHub notifies RepoGator every time an issue or pull request is opened.

03

Add your API keys

Paste your OpenRouter and OpenAI keys in Settings. They're stored encrypted and billed directly to your own account.

04

Let it run

That's it. No configuration, no pipelines, no maintenance. Connect a repo and walk away — RepoGator handles the rest.
